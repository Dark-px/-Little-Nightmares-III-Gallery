# Little Nightmares III Image Gallery

This is a simple image gallery website dedicated to the game "Little Nightmares III". It features a collection of images related to the game, presented in a clean and simple interface. The website also includes the main theme music from the game, which plays automatically in the background.

## How to View

To view the website, simply open the `index.html` file in your web browser.

## Technologies Used

* HTML
* CSS
